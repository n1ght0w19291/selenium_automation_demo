# selenium_automation_demo

## 簡介
此專案是一個 **Selenium 自動化練習範例**，示範如何透過環境變數與簡單配置來控制瀏覽器操作。  
您可以將此專案應用於 **學習網頁測試、自動化流程** 等情境。  

---

## 環境變數
請在專案根目錄建立一個 `.env` 檔案，內容格式如下（此為示意用，請自行修改）：  

```dotenv
PASSWORD=ABC123
ACCOUNT=U11116000
COURSE_CODE=329
DEBUG=false
```

---

## 使用方法
1. **安裝依賴：**  
   請先安裝 Python 套件：  
   ```bash
   pip install -r requirements.txt
   ```

2. **設定環境變數：**  
   確保 `.env` 文件存在於專案根目錄，並填寫正確資訊。  

3. **執行腳本：**  
   ```bash
   python ./src/main.py
   ```
